Struktur Kawalan

然而，代码并不只有一种格式。其包含了很多种写法，打些比方吧。

Struktur Kawalan Urutan (Jujukan)

顺序控制结构是最简单的控制结构，它没有分支，也没有条件判断。

举个例子，它可以以出生年份来计算你的年龄，因为其没有任何分支。

Input：获取用户的出生年份。
Process：用当前年份减去出生年份，得到年龄。
Output：显示计算出的年龄。



在这个例子中，每一步操作都按照顺序依次执行，没有任何分支或条件判断，这就是顺序控制结构的典型特点。

Struktur Kawalan Pilihan

1. Tunggal (单选)

单选控制结构使用条件判断来决定是否执行某个操作。

举个例子，如果用户输入的年龄大于18岁，则显示“你是成年人”。

Input：获取用户的年龄。
Process：检查年龄是否大于18。
Output：如果年龄大于18，则显示“你是成年人”。

记住，在单选中，若无触发条件，这不会有Output 


System.out.print("请输入你的年龄: ");
int age = scanner.nextInt();

if (age > 18) {
    System.out.println("你是成年人");
}

2. Dwipilihan (双选)

双选控制结构有两个分支，使用条件判断来决定执行哪个操作。

举个例子，如果用户输入的年龄大于18岁，则显示“你是成年人”，否则显示“你是未成年人”。

Input：获取用户的年龄。
Process：检查年龄是否大于18。
Output：如果年龄大于18，则显示“你是成年人”；否则显示“你是未成年人”。

与tunggal不同的是，这个不管怎样，都会有output 


System.out.print("请输入你的年龄: ");
int age = scanner.nextInt();

if (age > 18) {
    System.out.println("你是成年人");
}
else {
    System.out.println("你是未成年人");
}

3. Pelbagai Pilihan (多选)

多选控制结构使用多个条件判断来决定执行哪个操作。

举个例子，根据用户的成绩显示不同的等级。

伪代码:
Input：获取用户的成绩。
Process：根据成绩判断等级。
Output：显示对应的等级。


System.out.print("请输入你的成绩: ");
int score = scanner.nextInt();

if (score >= 70) {
    System.out.println("你的等级是A");
} else if (score >= 60) {
    System.out.println("你的等级是B");
} else if (score >= 50) {
    System.out.println("你的等级是C");
} else if (score >= 40) {
    System.out.println("你的等级是D");
} else {
    System.out.println("你的等级是F");
}

4. Switch-Case

Switch-Case控制结构最为特别，其不大像条件结构，其判断一个variable与一系列值（Case中的词典）中某个值是否相等，每个值称为一个分支。

举个例子，根据用户输入的月份显示对应的季节。

Input：获取用户输入的月份。
Process：根据月份判断季节。
Output：显示对应的季节。


System.out.print("请输入月份(1-12): ");
int month = scanner.nextInt();

String season;

switch (month) {
    case 1:
    case 2:
    case 12:
        season = "冬季";
        break;
    case 3:
    case 4:
    case 5:
        season = "春季";
        break;
    case 6:
    case 7:
    case 8:
        season = "夏季";
        break;
    case 9:
    case 10:
    case 11:
        season = "秋季";
        break;
    default:
        season = "无效的月份";
        break;
}

System.out.println("这个月份属于：" + season);

看看上面的代码，当一个variable精准中到了其中一个case（词典）的数值，则会执行它，没有的话会去default

Struktur Kawalan Ulangan

1. For

For 控制结构通过指定循环次数来重复执行一段代码。

举个例子，输出从1到10的数字：

Input：无
Process：使用循环输出数字。
Output：打印从1到10的数字。


for (int i = 1; i <= 10; i++) {
    System.out.println(i);
}

这段代码解释如下：
for 括号里的分别为 (Nilai mula ; Syarat ulang ; Syarat tambah nilai selepas satu ulangan)
Pengisytiharkan pembolehubah： `int i = 1;` - Isytihar一个整数变量 `i`，并将其值设为1。这个变量 `i` 用来追踪循环的当前状态。

Syarat： `i <= 10;` - 检查循环是否应该继续执行的条件。只要 `i` 小于或等于10，循环就会继续执行。

Penambahan： `i++` - 在每次循环结束后执行的操作。这会使变量 `i` 的值递增，i++ 的意思为 i=i+1

Output： `System.out.println(i);` - 在每次循环中输出当前 `i` 的值。因此，这段代码将会打印出从1到10的数字。

2. While

While 控制结构在条件为真(true)时重复执行代码块。

举个例子，输出从1到5的数字：

Input：无
Process：使用循环输出数字。
Output：打印从1到5的数字。


int i = 1;
while (i <= 5) {
    System.out.println(i);
    i++;
}

解释：当结束一轮轮回时，它会查回条件，若仍达成，即继续轮回
再次说下i++是i=i+1

和for不同的是，它可以是另一种呈现方式：
打个比方，收款机

收款机要求收银员输入金额直到完毕，才结算
我们无法预知顾客购买多少商品，因此不确定多少次轮回
这时，while可以做到这点，轮回直到用户停止，这是for做不到的。

3. Do-While

Do-While 控制结构首先执行代码块，然后检查条件是否为真，只要条件为真就重复执行。

举个例子，输出从1到5的数字：

Input：无
Process：使用循环输出数字。
Output：打印从1到5的数字。


        int i = 1;
        do {
            System.out.println(i);
            i++;
        } while (i <= 5);

解释：类似while，差别在于第一次不管真假，都会进行一次轮回先

Jenis-Jenis Data

1. [int] Integer (整数)

整数是没有小数部分的数字，可以是正数、负数或零。

举个例子，1, -3, 42 都是整数。

用途： 用于计数、循环计数器等。

代码示例：


// 声明整数变量
int number = 5;
System.out.println("整数: " + number);

2. [float] Float (浮点数)

浮点数是带小数部分的数字，用于表示更精确的数值。

举个例子，3.14, -0.001, 2.71828 都是浮点数。

用途： 用于需要精确度的计算，如科学计算、工程计算等。

代码示例：


// 声明浮点数变量
float pi = 3.14f;
System.out.println("浮点数: " + pi);

3. [double] Double (双精度浮点数)

双精度浮点数与浮点数类似，但有更高的精度和范围。

举个例子，3.141592653589793, -2.718281828459045 都是双精度浮点数。

用途： 用于需要更高精度的计算。

代码示例：


// 声明双精度浮点数变量
// 数值后有f
double pi = 3.141592653589793;
System.out.println("双精度浮点数: " + pi);

4. [char] Character (字符)

字符用于表示单个字符，如字母或符号。

举个例子，'a', 'Z', '@' 都是字符。

用途： 用于处理单个字符，如输入验证等。

代码示例：


// 声明字符变量
char letter = 'A';
System.out.println("字符: " + letter);

5. [String] String (字符串)记住String在Java大写！

字符串是一串字符的集合，用于表示文本。

举个例子，"Hello, World!", "Java 编程" 都是字符串。

用途： 用于处理文本数据，如用户输入、文件内容等。

代码示例：


// 声明字符串变量
String greeting = "Hello, World!";
System.out.println("字符串: " + greeting);

6. [boolean] Boolean (布尔)

布尔类型表示真(true)或假(false)两个值。

举个例子，true, false 都是布尔值。

用途： 用于条件判断、逻辑运算等。

代码示例：


// 声明布尔变量
boolean isJavaFun = true;
System.out.println("布尔值: " + isJavaFun);

Jenis Ralat

1. Ralat Sintaks

Ralat sintaks 发生在编程语言规则未被正确遵循时。

例如，代码书写错误或不正确使用标点符号。

解决方案： 检查代码以识别和修正语法错误。

示例代码：


// 语法错误示例
int x = 10
System.out.println("变量 x 的值: " + x);


// 语法正确示例
int x = 10;
System.out.println("变量 x 的值: " + x);

2. Ralat Logik

Ralat logik 发生在程序语法正确但行为不符合预期的情况下。

例如，数学计算产生不正确的结果或者不希望的输出。

解决方案： 检查算法逻辑以确保整个过程的正确性。

示例代码：


// 逻辑错误示例
int a = 5;
int b = 3;
int result = 5 - 3; 
System.out.println("加法结果: " + result);


// 逻辑正确示例
int a = 5;
int b = 3;
int result = 5 + 3; 
System.out.println("加法结果: " + result);

3. Ralat Masa Larian (时序错误)

Ralat Masa Larian 发生在程序计算是打破了数学的规则。

例如，任何数值除于0。

解决方案： 在程序使用资源之前，确保遵循数学规则。

示例代码：


// 逻辑错误示例
int a = 5;
int b = 0;
int result = a / b; // 这将导致逻辑错误（除以零）
System.out.println("除法结果: " + result);


// 逻辑正确示例
int a = 5;
int b = 1;
int result = a / b; 
System.out.println("除法结果: " + result);

Operator

Operator-operator Aritmetik (算术运算符)

Operator-operator aritmetik digunakan untuk melakukan operasi matematik dalam program.

Kepentingan:

Bahasa Melayu:

Membolehkan pengiraan dan manipulasi nombor dalam kod.

Chinese (简体中文):

用于在程序中执行数学运算和操作。

Operator Penerangan
+ Penambahan
- Pengurangan
* Perkalian
/ Pembahagian
() Grouping (Penjodohan)
% Modulus (Sisa bagi)
Contoh:


int hasil = 10 + 5; // Penambahan
System.out.println("Hasil penambahan: " + hasil);

int hasil2 = 20 - 8; // Pengurangan
System.out.println("Hasil pengurangan: " + hasil2);

double hasil3 = 4.5 * 2; // Perkalian
System.out.println("Hasil perkalian: " + hasil3);

double hasil4 = 10.0 / 3.0; // Pembahagian
System.out.println("Hasil pembahagian: " + hasil4);

int hasil5 = 15 % 4; // Modulus (Sisa bagi)
System.out.println("Hasil modulus: " + hasil5);

Operator Hubungan (关系运算符)

Operator hubungan digunakan untuk membandingkan nilai di antara dua operand.

Kepentingan:

Bahasa Melayu:

Digunakan untuk membuat keputusan berdasarkan perbandingan nilai.

Chinese (简体中文):

用于比较两个操作数的值。

Operator Penerangan
== Sama dengan
!= Tidak sama dengan
> Lebih besar dari
< Lebih kecil dari
>= Lebih besar atau sama dengan
<= Lebih kecil atau sama dengan
Contoh:


int a = 10;
int b = 20;

boolean sama = (a == b); // Sama dengan
System.out.println("Apakah a sama dengan b? " + sama);

boolean tidakSama = (a != b); // Tidak sama dengan
System.out.println("Apakah a tidak sama dengan b? " + tidakSama);

boolean lebihBesar = (a > b); // Lebih besar dari
System.out.println("Apakah a lebih besar dari b? " + lebihBesar);

boolean lebihKecil = (a < b); // Lebih kecil dari
System.out.println("Apakah a lebih kecil dari b? " + lebihKecil);

boolean lebihAtauSama = (a >= b); // Lebih besar atau sama dengan
System.out.println("Apakah a lebih besar atau sama dengan b? " + lebihAtauSama);

boolean kurangAtauSama = (a <= b); // Lebih kecil atau sama dengan
System.out.println("Apakah a lebih kecil atau sama dengan b? " + kurangAtauSama);

Operator Logikal (逻辑运算符)

Operator logikal digunakan untuk menggabungkan kondisi logikal antara dua atau lebih operand.

Kepentingan:

Bahasa Melayu:

Digunakan untuk menentukan kebenaran atau kefalsafahan suatu pernyataan.

Chinese (简体中文):

用于组合两个或更多操作数的逻辑条件。

Operator Penerangan
&& Logik AND
|| Logik OR
! Logik NOT
Contoh:


boolean kondisi1 = true;
boolean kondisi2 = false;

boolean danLogik = (kondisi1 && kondisi2); // Logik AND
System.out.println("Hasil dari kondisi1 && kondisi2: " + danLogik);

boolean atauLogik = (kondisi1 || kondisi2); // Logik OR
System.out.println("Hasil dari kondisi1 || kondisi2: " + atauLogik);

boolean tidakLogik = !kondisi1; // Logik NOT
System.out.println("Hasil dari !kondisi1: " + tidakLogik);

Operator Increment dan Decrement (增量和减量运算符)

Operator increment dan decrement digunakan untuk menambah atau mengurangkan nilai satu dari operand.

Kepentingan:

Bahasa Melayu:

Membolehkan peningkatan atau pengurangan nilai pembolehubah dengan cepat.

Chinese (简体中文):

用于快速增加或减少变量的值。

Operator Penerangan
++ Increment (Peningkatan)
-- Decrement (Pengurangan)
Contoh:


int x = 10;
x++; // Increment x (x = x + 1)
System.out.println("Nilai x selepas increment: " + x);

int y = 5;
y--; // Decrement y (y = y - 1)
System.out.println("Nilai y selepas decrement: " + y);

Subaturcara

Subaturcara merujuk kepada bahagian dalam program yang terpisah untuk melaksanakan tugas tertentu secara berulang.

Kepentingan:

Bahasa Melayu:

Subaturcara memainkan peranan penting dalam pengaturcaraan moden dengan memisahkan kod ke dalam bahagian-bahagian yang kecil dan berfungsi khusus. Ini tidak hanya meningkatkan keseluruhan struktur dan keterbacaan kod, tetapi juga memudahkan penyelenggaraan dan pengurusan projek.

Chinese (简体中文):

小程序指程序中的独立部分，用于重复执行特定任务。小程序在现代编程中扮演着重要角色，将代码分成小而专一的部分不仅提高了整体结构和代码的可读性，还便于项目的维护和管理。

Prosedur (程序)

Prosedur adalah subaturcara yang tidak mengembalikan nilai (void) dan digunakan untuk menjalankan tugas tertentu dalam program. Prosedur biasanya digunakan untuk menjalankan operasi yang berulang dengan struktur yang teratur.

Kepentingan:

Bahasa Melayu:

Prosedur membantu mengelompokkan pernyataan-pernyataan yang berfungsi sama ke dalam satu unit logik. Ini membantu dalam menyusun program dengan lebih teratur dan mengurangkan pengulangan kod yang tidak perlu.

Chinese (简体中文):

程序是一种不返回任何值的子程序，用于在程序中执行特定的操作。程序通常用于执行重复性的操作，并将具有相同功能的语句组织在一起，有助于更有序地编写程序并减少不必要的代码重复。

Contoh:

void prosedurKataSalam() {
    System.out.println("Hello, World!");
}

// Memanggil prosedur
prosedurKataSalam(); // Output: Hello, World!

Fungsi (函数)

Fungsi adalah subaturcara yang mengembalikan nilai dan dapat dipanggil dari tempat lain dalam program. Fungsi biasanya digunakan untuk menghitung nilai kembali berdasarkan logik atau perhitungan tertentu.

Kepentingan:

Bahasa Melayu:

Fungsi memungkinkan kita untuk menulis kod yang dapat digunakan kembali dengan cara yang lebih efisien. Ini memisahkan logik perhitungan dari bagian utama program, memudahkan pengujian dan perubahan kode, serta meningkatkan modularitas.

Chinese (简体中文):

函数是一种返回值的子程序（会返回数值），可以从程序的其他地方调用。函数通常用于基于特定的逻辑或计算返回值。

Contoh:

// Fungsi untuk melakukan penambahan
int tambah(int a, int b) {
    return a + b;
}

// Memanggil fungsi dan menampilkan hasilnya
int hasilTambah = tambah(5, 3);
System.out.println("Hasil tambah: " + hasilTambah); // Output: Hasil tambah: 8

Tips


// Memanggil fungsi dan menampilkan hasilnya
int hasilTambah = tambah(This is [Argument])
System.out.println("Hasil tambah: " + hasilTambah); // Output: Hasil tambah: 8


// Fungsi untuk melakukan penambahan
public static int tambah(This is [Parameter]) {
    return a + b;
}

Langkah-langkah Penyelesaian Masalah

Fokus on penerangan

1. Mengumpulkan dan Menganalisis Data

Murid perlu mengumpul data tentang punca dan skop masalah serta mengenal pasti hal yang berikut dalam sesuatu situasi:

  • Sejauh manakah masalah tersebut memberikan kesan?
  • Bilakah masalah itu perlu diselesaikan?

2. Menentukan Masalah

Murid perlu mengenal pasti masalah yang paling utama yang perlu diselesaikan. Kemudian, masalah yang kedua juga perlu dikenal pasti sebelum mengenal pasti tahap pemahaman sedia ada murid.

  • Adakah masalah ini merupakan masalah yang utama?
  • Apakah masalah lain yang dapat dikesan?
